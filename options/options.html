Works with the Window Titler extension: the name it shows in the window title is offered as the default session name when you start tracking a window. When disabled (or when no preface is found), you are simply asked to type a name.
Writes “[Session name]” into the window title when you track, rename, or restore a session — the same mechanism Window Titler uses, so restored windows keep their visible name. Caution if you actively use Window Titler on the same windows: the two extensions overwrite each other (last writer wins), and Window Titler may reapply its stored name after a browser restart.
Every new window becomes a session without asking. The name is taken from the window title preface when one appears (Window Titler), otherwise a timestamped name is used — rename it later in the sidebar. Private windows are never tracked.
Sessions are already saved automatically whenever tabs change (open, close, navigate, pin, move). This adds a periodic safety-net snapshot. Set to 0 to rely on tab-change saves only.
Export downloads all sessions (open ones included, as snapshots) to a JSON file. Import merges sessions from such a file as closed sessions; re-importing the same file does not create duplicates.
